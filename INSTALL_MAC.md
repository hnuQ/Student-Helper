# Student Helper macOS 安装指南

本软件为开源免签名版本。安装时请执行以下操作以绕过 macOS 安全拦截：

### 方法 1：右键开启（推荐）

1. 将应用拖入"应用程序 (Applications)"文件夹。
2. 首次打开时，**请勿直接双击**。请在应用图标上**右键点击**，选择**"打开"**。
3. 在弹出的系统警告框中，再次点击"打开"或"仍要打开"。

### 方法 2：终端强制解除（若提示"已损坏"）

如果系统提示"文件已损坏"或"无法验证开发者"，请执行以下操作：

1. 打开 Mac 自带的"终端 (Terminal)"。
2. 复制并运行以下命令（可能需要输入电脑开机密码）：

   `sudo xattr -rd com.apple.quarantine "/Applications/Student Helper.app"`

3. 执行完毕后，即可正常双击打开软件。
